# 0100相同的树

## 问题描述

给你两棵二叉树的根节点 p 和 q ，编写一个函数来检验这两棵树是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

 

示例 1：

输入：p = [1,2,3], q = [1,2,3]
输出：true

示例 2：

输入：p = [1,2], q = [1,null,2]
输出：false

示例 3：

输入：p = [1,2,1], q = [1,1,2]
输出：false

## 解法

随便什么遍历方式都可以，只要遍历顺序一样就行。细节太多了，主要包括空指针的判断，空指针的处理，节点相同和不相同情况的处理
